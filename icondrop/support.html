IconDrop Support

IconDrop is a simple macOS app for creating beautiful folder and file icons from your own images.

What can IconDrop do?

IconDrop lets you:

  • Drag in a PNG, JPG, JPEG, or HEIC image
  • Preview several Mac-style icon designs
  • Export icons as PNG or ICNS
  • Apply generated icons directly to folders
  • Unlock export and folder application with one lifetime purchase

Supported Image Formats

IconDrop supports:

  • PNG
  • JPG / JPEG
  • HEIC

How to Apply an Icon to a Folder

  1. Open IconDrop.
  2. Drag in an image or click “Choose Image”.
  3. Select an icon style.
  4. Click “Apply to Folder”.
  5. Choose the folder you want to customize.
  6. The folder icon will be updated in Finder.

If Finder does not refresh immediately, try closing and reopening the Finder window.

Purchases

IconDrop is free to download. You can import images and preview styles for free.

Exporting PNG / ICNS files and applying icons to folders requires a one-time lifetime unlock.

There is no subscription and no advertising.

Restore Purchase

  1. Open IconDrop.
  2. Click Export or Apply to Folder.
  3. Choose “Restore Purchase” in the unlock window.

Troubleshooting

The folder icon did not update

Try:

  • Closing and reopening the Finder window
  • Switching Finder view mode
  • Choosing the folder again
  • Making sure the folder is not locked or restricted

The image will not import

Make sure the image is PNG, JPG, JPEG, or HEIC.

Export does not work

Export requires IconDrop Lifetime Unlock.
